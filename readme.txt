Implementation of the LESS (Leaner CSS) in order to make CSS development easier.


== Description ==
All themers can save their code with <code>*.less</code> files so as they are automatically parsed and get ready for caching/compression/shrinking with other plugins. 


**Minimal Requirements**: PHP 5.1.2 and WordPress 2.8.
